# Security Policy

## Supported Versions

| Versão | Suporte |
| :--- | :--- |
| 0.9.x | ✅ Ativo |
| < 0.9 | ❌ Sem suporte |

## Reporting a Vulnerability

Caso identifique uma vulnerabilidade de segurança no H.E.L.I.O.S.:

- **Não abra uma issue pública** com detalhes da vulnerabilidade
- Entre em contato diretamente por e-mail (matheus.lim2006@gmail.com) antes de qualquer divulgação
- Aguarde confirmação antes de publicar qualquer informação

Vulnerabilidades confirmadas serão corrigidas e documentadas no [`CHANGELOG.md`](./CHANGELOG.md).
