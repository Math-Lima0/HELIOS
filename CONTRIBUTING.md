# Contributing

H.E.L.I.O.S. é atualmente um projeto de desenvolvimento individual,
parte da iniciativa Pantheon Systems.

Contribuições externas não estão abertas neste momento.
Se você encontrou um bug ou tem uma sugestão técnica,
abra uma [issue](../../issues) descrevendo o problema e o contexto.

Para colaborações acadêmicas ou profissionais, entre em contato diretamente.
